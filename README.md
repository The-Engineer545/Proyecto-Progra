# Moto Wheelie 

Juego de motos en consola con:
- Llanteo (turbo x3)
- Dificultad progresiva
- 2 cajas por spawn (una obligatoria en tu altura)
- Ranking persistente en JSON
- Menú con flechas
- Arquitectura modular (14 archivos .h/.cpp)

## Controles
- W/S: subir/bajar
- ESPACIO: llanteo
- ESC: salir
